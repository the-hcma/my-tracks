📱 Configure OwnTracks App

Set up your phone to send location data to this server:

1. Install OwnTracks

Download from Google Play (Android) or App Store (iOS)

2. Open Preferences

Tap ☰ menu → Preferences → Connection

{% if mqtt_tls_enabled %}

3. Choose Connection Mode

MQTT TLS (Recommended): Encrypted, client certificate authentication, best security

{% if mqtt_enabled %}

MQTT: Better battery life, real-time updates

{% endif %}

HTTP: Simpler setup, works without persistent connection

4a. For MQTT TLS Mode (Recommended):

Mode: MQTT

Host: {{ hostname }}

Port: {{ mqtt_tls_port }}

TLS: ON

Authentication: ON (use your server username/password)

Import the CA certificate and your client certificate (.p12) onto your device. See your Profile to download them.

{% if mqtt_enabled %}

Mode: MQTT

Host: {{ hostname }}

Port: {{ mqtt_port }}

TLS: OFF

Authentication: ON (use your server username/password)

{% endif %}
